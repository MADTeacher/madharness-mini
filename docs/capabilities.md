# Возможности ветки

`02-AGENTS-md` расширяет минимальный harness двумя идеями:

- модель получает правила проекта из `AGENTS.md`;
- модель может попросить инструмент `read_image`, если vision input включён в
  конфиге.

## Команды CLI

| Команда | Что делает |
| --- | --- |
| `madharness-mini init` | Создаёт или обновляет `.madharness-mini/config.json`. |
| `madharness-mini ask "..."` | Отправляет один запрос модели без инструментов, но уже с учётом `AGENTS.md`. |
| `madharness-mini run "..."` | Запускает агентский цикл с инструментами workspace. |
| `madharness-mini trace <id>` | Показывает краткую сводку JSONL-трассы. |

## Инструменты режима `run`

| Инструмент | Что делает |
| --- | --- |
| `list_files` | Показывает файлы workspace без служебных каталогов. |
| `read_file` | Читает UTF-8 фрагмент файла. |
| `write_file` | Полностью перезаписывает UTF-8 файл. |
| `search_code` | Ищет буквальную подстроку в проекте. |
| `apply_patch` | Применяет точечные файловые правки. |
| `run_shell` | Запускает одну безопасно разобранную команду. |
| `read_image` | Прикрепляет локальное изображение к следующему запросу модели. |

## Настройки

К базовым настройкам первой ветки добавились поля для изображений:

| Поле | Смысл |
| --- | --- |
| `supports_image_input` | Разрешает `read_image` передавать изображение модели. |
| `max_image_bytes` | Ограничивает размер файла изображения. |
| `image_detail` | Передаётся в мультимодальный payload как detail. |

Если `supports_image_input=false`, `read_image` возвращает отказ. Это нужно,
чтобы пользователь явно выбрал модель, которая умеет работать с изображениями.

## Проектные инструкции

`AGENTS.md` читается из корня workspace и вложенных директорий до текущей cwd.
Текст добавляется в system prompt после встроенных правил harness. Суммарный
лимит защищает prompt от случайно огромных инструкций.

В этой ветке ещё нет отдельного `ContextManager`: prompt собирается прямо в
`loop.py`. Это важная учебная точка, потому что следующая ветка покажет, почему
такую сборку полезно вынести в отдельный слой.
